## Q14. （簡答）為什麼貪狼星不喜歡和文昌或者文曲同宮？

!!! note "A14"

## Q15.（簡答）解讀星曜的時候需要注意什麼？

!!! note "A15"
    答案：先從主星開始；雙星的，先解釋前面的主星，再解釋後面那個主星；空宮，要看對宮，能借的時候表示表裡如一，內外一致，不能借時，要解釋對宮，同時要知道是被哪個星卡住不能借的；解釋完主星，再加上輔星、四化；一條一條，並且要注意始終圍繞主星來解釋。
