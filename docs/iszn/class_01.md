## 50初階01


# 第一堂筆記

20260416 50屆 初階 


紫微斗數中的星曜最早出現在古書中北斗七星延命經，包含在佛教大藏經內。
延命的意思是如何讓生命更好。
**命理學意即人與環境的關係。*

紫微斗數起源在明清兩代，皇室集合當時的命理學，為欽天鑑使用的綜合學，前身叫做七政四餘。

命盤上有十二空格，中間是命主命盤資料，外圍的是十二宮，地支位是固定的，子丑寅卯辰巳午未申酉戌亥，對應十二生肖，也是基礎計算時間的方法。
天干是變動的。

紫微斗數裡有兩大主要體系，一個是宮位體系、一個是星曜體系；
宮位代表你在甚麼樣的環境，星曜代表你會扮演甚麼樣的角色/做甚麼事情

*十二宮介紹
宮位對面叫做對宮，每個宮位都要看對宮
每個宮位對宮就是代表該宮位內心跟外在表現
(順時鐘排列 : 命宮、父母宮、福德宮、田宅宮、官祿宮、僕役宮、遷移宮、疾厄宮、財帛宮、子女宮、夫妻宮、兄弟宮)

1.命宮-遷移宮:
1.1 命宮管十二宮。你對生命價值看法、態度，會影響你其他的宮位
1.2遷移宮是在外面展現的樣子，亦代表命主自己的內心

2.兄弟宮-交友宮
2.1兄弟宮代表同性別兄弟姊妹
2.2交友宮代表不同性別兄弟姊妹
2.3本命盤中的兄弟姐妹，只能是兄姊，因為在出生時候就已經存在，而且要有一個程度的年齡差異 。

3.夫妻宮-官祿宮
3.1夫妻宮代表對感情態度，並非老公老婆或情侶
3.2.官祿宮代表生活重心的追求，並非職業

4.子女宮-田宅宮
4.1子女宮代表所有跟生小孩有關的事情，例如 : 教養小孩態度、性生活等
4.2田宅宮代表對家的看法、與家人的關係、居住的環境

5.財帛宮-福德宮
5.1財帛宮代表賺錢的能力、對錢的看法、對錢如何花用
5.2算命時，很多人問職業，其實是在問賺錢，賺錢就要看財帛宮
5.2福德宮代表精神狀態與福報運氣

6.疾厄宮-父母宮
6.1疾厄宮代表與生俱來的長相、樣貌與疾病
6.2父母宮代表父親，以及遺傳。

根據出生時的本命盤能論斷的只有父親跟母親，父親看父母宮，母親則看兄弟宮。
父母宮的夫妻宮=兄弟宮代表媽媽

命盤講的是實質影響力。
若出生時父親已不在，那麼，本命盤父母宮位只能看遺傳。
若有其他長輩撫養，也必須是命主認定此人為父親角色跟地位，才能以父母宮來論定。
EX: 養父或繼父。

本命盤看的是與生俱來的特質，因此，如果要看感情狀態，必須加上時間因素 (大限十年或流年)，再去找該對應夫妻宮，才是看當下的感情對象

*四煞星:火鈴羊陀 ，煞星是人的潛意識，皆為輔星，影響宮位也影響主星。影響可以同時存在。
1.火星:來得快去得快、熱情、衝動、不加思考、持續力不足
2.鈴星:有謀略、有計畫去做、冷靜、算計、盤算
3.擎羊:決斷，決策通常是經過思考過的
4.陀羅:糾結而不知如何選擇，被困在原地

*四化: 因星曜而產生，必須配合星曜解釋，非單獨存在
1.化祿:本來不是你的，而多出來的東西
2.化權:掌握、掌控，也代表雙
3.化科:彰顯、表現、被大家看見
4.化忌:有所空缺，因此也會去追求對宮的事

*子女宮&田宅宮代表財庫，基於有土斯有財及養兒防老的概念

*命盤時間以當地時間計算
