# 第三堂課後作業


## Q1.（簡答題）

 1. 十天干及十二地支，請分別說明其陰陽屬性為何？
 1. 天干搭配地支計年時，陽天干必須搭配何種陰陽屬性之地支？

!!! note "A1"
   （1）

    陽天干：甲丙戊庚壬

    陰天干：乙丁己辛癸

    陽地支：子寅辰午申戌

    陰地支：丑卯巳未酉亥

   （2）陽地支



## Q2.（簡答題）

西元年尾數為9，其天干為何？

!!! note "A2"
    天干為己

## Q3.（簡答題）

蜜柚是個小男生，2023 年出生，請問：

 1. 他的生年天干為何？
 1. 他是陽男還是陰男？
 1. 大限為順時針走還是逆時針走？
 1. 2024 年他的流年命宮所在地支位為何？

!!! note "A3"
（1）天干為癸。

（2）陰男。

（3）大限為逆時針走。

（4）辰位。


## Q4.（簡答題）

七殺、破軍、貪狼必在三合內。
殺破狼組合中：

 1. 其中哪顆星是起始點，代表慾望的發生？
 1. 其中哪顆星是力量，代表堅忍的意志力？
 1. 其中哪顆星是最終的追求，代表夢想？

!!! note "A4"

    （1）貪狼

    （2）七殺

    （3）破軍


## Q5.（簡答題）

請解釋何謂正財？

!!! note "A5"
    透過努力而獲取穩定利益，可以擬定計畫執行且有預期收益，其作法與當下法律規範無關。


## Q6.（簡答題）

武曲需要放在哪兩個宮位才明確指的是金錢？

!!! note "A6"
    命宮、財帛宮


## Q7.（簡答題）

三大財星分別為三個星曜？

!!! note "A7"
    武曲、天府、太陰。


##Q8.（簡答題）

請解釋武曲化祿的含義

!!! note "A8"
    因為務實、腳踏實地的價值觀或做事態度，而得到額外的好處。


## Q9.（單選題）

下列何者「不是」武曲喜歡跟桃花星放在一起的理由？

  A. 能夠增加桃花人緣
  B. 能夠讓武曲在實事求是的態度上，還能發揮創意
  C. 能夠讓武曲更加冷靜

!!! note "A9"
    C


## Q10.（簡答題）

武曲不喜歡跟火星、擎羊、陀羅破軍放在一起。理由為何？

!!! note "A10"

    武曲跟火星在一起，容易因衝動亂花錢，

    跟擎羊在一起，容易因固執亂花錢，
    
    跟陀羅在一起，花錢猶豫不決，

    跟破軍在一起，容易因夢想亂花錢，導致資源應用上容易出問題。



## Q11.（選擇題）

 A. 骨架粗大

 C. 皮膚白

!!! note "A11"
    C


## Q12.（簡答題）

請定義何謂身宮？

!!! note "A12"
    身宮所在的宮位，代表一生所重視的價值，並且會在能夠獨立自主生活的時候才顯現。


## Q13.（簡答題）

身宮只會出現在哪個命盤？

!!! note "A13"
    本命盤


## Q14.（填空題）

身宮在官祿宮不一定代表工作狂，因為身宮要對應宮內_____來解釋。

!!! note "A14"
    星曜


## Q15.（簡答題）

三方四正中，命宮是「我的主軸」，所以代表我的主要意志。
遷移宮代表外在執行。

 1. 財帛宮代表什麼？
 1. 官祿宮代表什麼？

!!! note "A15"
（1）財帛宮代表資源、金錢來源。

（2）官祿宮代表工作表現、行動方向。


## Q16.（簡答題）

請解讀紫微星在自己本命盤宮位所代表的意義。

若紫微星在本命盤疾厄宮，則改看大限盤宮位。
紫微天相組合可以跳過，或者練習本命盤命宮紫微七殺、遷移宮天府的組合。
