# 第八堂課後作業

## Q1.複選題：下列描述雙胞胎命盤何者正確？

（A）出生時所排出來的命盤為老大的本命盤，該命盤的遷移宮作為老二本命盤的命宮。
（B）大限的運限宮干相同，遇到的四化及祿羊陀也相同。
（C）大限的運限宮干不同，遇到的四化及祿羊陀也不同。
（D）大限的運限走向不分男女，一律跟著老大的走向。

!!! note "A1"
   （A）（C）


## Q2.選擇題：姊姊和弟弟是丙年生的雙胞胎，姊姊命宮坐酉位，弟弟命宮坐卯位，請問第三大限姊弟倆大限命宮分別坐在哪個地支位？

（A）姊姊亥位，弟弟丑位
（B）姊姊亥位，弟弟巳位
（C）姊姊未位，弟弟巳位
（D）姊姊未位，弟弟丑位

!!! note "A2"
    A2.（C）
    丙年為陽天干，陽男大限順時針走，陽女大限逆時針走，姊姊第三大限會坐落在未位，弟弟第三大限會坐落在巳位。
