# 第七堂課後作業

## Ｑ１　簡答題：若某位命主於西元1999年7月出生，請問其生年「天喜星」落於哪一個地支位？

!!! note "Ａ"
    午位


## Ｑ２　多選題：請問「紅鸞星」與「天喜星」會出現在哪幾張命盤上？ 
（Ａ）本命盤 （Ｂ）小限盤 （Ｃ）大限盤 （Ｄ）流年盤

!!! note "Ａ"
    Ａ、Ｄ

Ｑ３　簡答題：天同星的「化氣」為何？又稱為什麼「主」？
Ａ： 化氣為福，為福德主。


Ｑ４　選擇題：下列關於「天同星」的描述，何者錯誤？ 
（Ａ）天同星個性樂觀天真、看得開。 
（Ｂ）天同星具有博學的特質。 
（Ｃ）天同星適合與火星、鈴星、擎羊、陀羅等煞星同宮或會照。 
（Ｄ）具備福星特質，故天同星較不怕煞星，但不宜會見過多煞星。
Ａ： Ｃ

Ｑ５　簡答題：請問哪兩顆星曜同宮時，會具備「遇難呈祥」的特質？
Ａ： 天同星與天梁星（同梁同宮）。


Ｑ６　簡答題：承上題，引發此特質的條件為何？
Ａ： 當運限的命宮走到「天同、天梁同宮」的位置時。


Ｑ７　簡答題：某位男性命主於西元2030年出生，其本命盤的命宮在亥位，且天同、天梁同宮於寅位。請問他在第幾大限時，最有可能發生「遇難呈祥」的狀況？
Ａ： 第四大限。

Ｑ８　簡答題：為何老師提到「太陰、天同」在午位時，會比在子位更適合創業當老闆？
Ａ： 因為個性較不受框架約束。

Ｑ９　簡答題：巨門星的「化氣」為何？又稱為什麼「主」？
Ａ：化氣為暗，為品物主。
時間軸： 02:16:50、02:35:01

Ｑ１０　簡答題：看巨門星時，需特別注意哪一顆星曜的「旺弱」位置？
Ａ： 太陽星。

Ｑ１１　選擇題：請問下列哪一種星曜組合，會讓巨門星的不安全感變得較為隱性？ 
（Ａ）巨門星在未位，太陽星在酉位 
（Ｂ）巨門星在午位，太陽星在戌位 
（Ｃ）巨門星在巳位，太陽星在亥位 
（Ｄ）巨門星在寅位，太陽星在寅位
Ａ： Ｄ


Ｑ１２　簡答題：課堂中所述的「明日驅暗格」，需要巨門星與太陽星分別落於哪一個地支位才會成立？
Ａ： 巨門在亥，太陽在巳。


Ｑ１３　簡答題：巨門星若要成為「庫星」，需具備哪些條件？
Ａ： 太陽處於旺位 ＋ 巨門化祿（或遇祿存） ＋ 落於財帛宮。


Ｑ１４　簡答題：請根據以下特質判斷，命主可能是由哪兩顆主星坐命？
個性自得其樂，較不喜與人接觸，喜歡獨自鑽研事物。
博學有才華，不安全感較重，有囤積物品的習慣。
聰明有條理，邏輯能力佳，學生時期成績優異。
Ａ： 天機星與巨門星（機巨同宮）。


Ｑ１５　選擇題：根據課堂所述，巨門星具有「黑洞」的特質，請問它會吞噬掉下列哪一顆星曜？ 
（Ａ）太陽 （Ｂ）天同 （Ｃ）擎羊 （Ｄ）文昌
