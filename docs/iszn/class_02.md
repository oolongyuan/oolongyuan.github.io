# 第二堂筆記

## 《前導課》

**大限** ：  
:  這十年的個性價值及現象。

   以實歲計，十年為一個單位，  在命盤中對應的十年歲數區間宮位作為大限命宮

**小限** ：  
:  今年對事情作出的決定及選擇。
   
   看虛歲

**虛歲及實歲算法**：

 ─ 實歲算法：現在年份減去出生年份(※ 需注意出生年農曆年份)

```
  例：西元2022年1月1日出生，因出生時農曆還在2021年，
      實歲為2026-2021=5歲
   
      ─ 虛歲算法：實歲+1

  例：以前述例子來看，虛歲則為5+1= 6歲。

      ※  紫微攻略裡的大限用『實歲』，
         文墨天機的大限用的是『虛歲』。
```

## 《正課》

**命理學的概念是推理** ：  透過條件的堆疊，一步一步推理去比對各種可能性，而推論必須合乎邏輯結果，重點是學習『**推論的過程、理解背後的原因**』而非背誦單一答案。

學習任何一顆主星時，需要記星曜重點：

 - 1、名稱：如“紫微”、“天機”等。
 - 2、五行：每顆星曜都有五行屬性(金、木、水、火、土)。
 - 3、化氣為何：星曜最根本的核心價值，
 - 4、主星所有的解釋須對應宮位 無法單獨存在。

### 紫微星

 - 五行屬『陰土』，**化氣為尊**： 尊貴、地位、尊嚴、面子、自尊心之所在。
 - 代表內心對“尊嚴”的追求。紫微星落在哪個宮位，就表示命主希望在那個領域獲得**尊重、體現價值**。
 - 需要的條件： 三方四正內要碰到*左輔、右弼、天魁、天鉞、天府、天相*，湊到越多(同時要注意天府天相所在宮位的狀態)，越能呈現紫微的優勢特質，吉星在旁邊照會，代表有人幫忙、有貴人扶持，能真正落實其“**尊貴**”的地位。
 - 長相貴氣，天庭飽滿，額頭高，髮際線呈圓形，柳葉眉、眉毛偏淡，中年後容易有小肚子，顴骨偏高但不突出，易有法令紋。

> 全部湊到稱『君臣慶會格』。

### 空宮

 - 宮位內無主星，可借對宮的全部主星(包含對宮主星的四化 )。
 - 當空宮內出現『**文昌、文曲、擎羊、陀羅、火星、鈴星**』中的任意一顆，則不能借對宮主星 。
 - 不能借時的解讀方法：*用這六顆星的特質搭配宮位解釋*。

### 六吉星

 - **左輔、右弼、文昌、文曲、天魁、天鉞**
 - 共同作用：幫助主星。但“幫助”不一定是好事，需結合宮位和主星性質判斷。

**△文昌**：
 
 - 代表理性、規範、主流價值的思維與行為模式。主科甲、文書。
 - 指正途功名（符合社會期待）。

**△文曲**：

 - 代表感性、創意、非主流價值的思維與行為模式。
 - 主才藝、巧思。指異途功名（憑藉特殊才藝）。

**△左輔 、右弼**：

 - 代表平輩、無差別的幫忙。
 - 不論對錯，你要幫我就幫。
 - **左輔**：代表直接、明顯的男性平輩的幫助。
 - **右弼**：代表細膩、私底下的女性平輩的幫助。
 - 在本命盤：表示命主自己具有樂於助人的特質。
 - 運限盤（大限/流年）：表示該時期真的會出現這樣一個平輩幫手。

**△天魁 、天鉞**：

 - 代表長輩、有原則的幫忙。
 - 通常基於“**正確價值觀**”提供協助或提攜。
 - **天魁**：代表陽貴人，偏向檯面上的幫助。
 - **天鉞**：代表陰貴人，偏向私底下的幫忙。
 - 本命盤代表自己是這樣的人，運限盤代表會出現這樣的貴人。

> 補充雙主星：當兩顆主星同宮時，它們會相互影響，排在前面的星曜為主     、後面的為輔。     輔星則會同時影響兩顆主星。

### 天府星

 - 五行屬『陽土』，**化氣為權**。
 - 務實且有謀略 ，同時是紫微斗數中的『財星』和『福星』，可以**化煞為用**(煞星不會不見，是被天府拿來使用)，但三方四正之內不可以超過兩顆煞星。
 - 天府*需要*祿存星，同宮才算（或與化祿之主星同宮）才能真正累積財富。
 - 和紫微星呈對角線排列組合。
 - 額頭方正，劍眉、下巴偏方。

### 祿存星

 - **乘旺主星**：只乘旺同宮主星，會放大主星的優勢，而讓主星缺點不明顯，但並不表示缺點不存在或減少。
 - 當同宮沒有主星時，會對該宮位*產生不安全感*。
