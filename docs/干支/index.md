# 天干地支

**天干地支** 是十干與十二支的合稱、簡通稱為十天干十二地支。

由兩者經一定的組合方式搭配成六十對，為一周期，循環往復，稱為一甲子或花甲之年。

干支之實，可以上追至出土商代甲骨文。
干支之名，本來叫幹枝，上追至東漢班固白虎通德論。
論商湯名時，曰甲乙者幹也，子丑者枝也，好比樹幹樹枝[1][2]，有主幹分枝之意，後來簡寫成干支。

每個天干和地支都代表著不同的意義，在命理學、曆法、風水等領域都有應用。

其中，中國古代用**干支**來*紀年、紀月、紀日、紀時*。 

## 天干

天干有十個，分別是：甲、乙、丙、丁、戊、己、庚、辛、壬、癸。 

### 天干五合

**天干五合**是指十天干中，按照特定顺序的五对相合关系：
*甲己合化土、乙庚合化金、丙辛合化水、丁壬合化木、戊癸合化火*。

这种组合是阴阳两气相合产生的一种变化，在传统命理学中用于分析五行力量的动态变化。 

#### 理论来源

天干五合的理论起源可以追溯到《河图》和《黄帝内经·素问》等古籍。 

河图：河图中的“一与六”、“二与七”、“三与八”、“四与九”、“五与十”的对应关系，是天干五合的理论基础。

黄帝内经：书中记载的“岁运说”也揭示了天干合化的对应关系，如“甲己之岁，土运统之”等。
