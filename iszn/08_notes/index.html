    天相星 廉貞星

    第八堂筆記

    前導課 : 雙胞胎

    1. 基本狀況：

      1. 小時候的價值觀、個性、長相差不多( 第一大限 : 命遷線、父疾線相同)。
      2. 家世背景差不多( 第一大限 : 子田線原生家庭相同)。
      3. 隨著年紀越來越大，他們的價值觀、個性、想法、作法就會慢慢的越來越不同(因為每個大限的宮干不一樣)。

    2. 多胞胎：

      1. 雙胞胎老大 : 使用出生時辰起命盤。
      2. 雙胞胎老二 : 使用老大的遷移宮為命宮，依序排出12宮。
      3. 龍鳳胎 : 使用老大的遷移宮為命宮，依序排出12宮，大限陽男陰女順走、陰男陽女逆走。
      4. 多胞胎老三 : 以老大的兄弟宮當命宮，依序排出12宮。
      5. 多胞胎老四 : 以老大的僕役宮當命宮，依序排出12宮。
      6. 雙胞胎小限(小限:男順女逆) : 出生年的生肖(地支)，兩人會一樣，所以兩人的小限一歲是一樣的。

    (補充)宮位：辰、戌、丑、未(四墓地)，不會有祿存進來。

    身宮的位置：

    依出生的時辰，安在本命盤上六個不同的宮位。

    1. 子/午時：身宮在命宮。
    2. 丑/未時：身宮在福德宮。
    3. 寅/申時：身宮在官祿宮。
    4. 卯/酉時：身宮在遷移宮。
    5. 辰/戌時：身宮在財帛宮。
    6. 巳/亥時：身宮在夫妻宮。

    【正課】

    天相星

    1. 化氣為印。

      1. 印是指承諾、規矩、規範。

      2. 太陽 vs天相

        • 太陽：由我制訂規範，大家都要聽我的。
        • 天相：天相星的印，主要是在人際關係上，會有自己的生活規範，並在這規範內去運作。

    2. 五行屬陽水(五行屬水的都是桃花星)。

      天相星的桃花偏向廣泛的人際關係，而重視人際關係也比較容易有好的異性關係。

    3. 官祿主 (基本人物代表：宰相)

      1. 天相為宰相，與帝星(紫微和天府)相同，喜歡三方四正有吉星幫忙。
      2. 宰相要做的是去協調皇帝跟臣子之間的關係，要在各部門之間斡旋，讓大家會聽我的、相信我的話、願意給我面子，因為我有辦法給一個承諾出來，這就是天相在做的事情。

    ……………………………………………………………………

    星曜排列位置

    • 紫府有一定的排列位置(初階第2堂內容)。
    • 天府的對宮一定是 : 七殺。
    • 殺、破、狼順時針走，一定在三合內。
    • 破軍的對宮一定是天相星。
    • 紫微的官祿宮固定是廉貞星、財帛宮是武曲星、兄弟宮是天機星。
    • 天機的夫妻宮固定是太陽星。
    • 太陽的夫妻宮固定是天同星。
    • 天相的父母宮一定是天梁星。
    • 巨門的父母宮一定是天相。
    • 天梁星的夫妻宮一定是巨門星。
    • 巨門星的福德宮一定是天梁、夫妻宮是太陰星。

    天相星的對宮一定是破軍星。

    對宮(破軍)就是他的內心跟展現：這個人很有規矩，內心有很多夢想，所以會希望把每件事都做得很好。

    為什麼天相怕煞忌？

    天相本來應該把事情安排得很好，結果衝動(情緒)了，就不容易安排好。本來會很在乎人際關係，在乎做事情要有條理，因為要達成內心的夢想，自然不會是個不好相處的人，但如果糾結、算計或太衝動，就會破壞這種情況。

    天相化忌，化忌就是因為規則而產生空洞。

    例如 :「為什麼每次出去都是我付錢」，「為什麼你常遲到？」雖然遲到只有兩分鐘而已，但我的規矩是不准遲到，我會去計較這件事，這是化忌的展現。 換言之，天相怕遇煞忌，是因為這會讓他的秩序和規範不照一般世俗邏輯走，但對天相來說，認為還是在規範他的秩序，只是外界不這麼覺得而已。

    約定的事被破壞稱為官非，與官非有關的星曜有天相和文昌，二者有何不同?

    所謂的官非指的是「約定被破壞」，而不是真的要去坐牢或犯法。因為歷朝歷代的狀況不同，所以這裡指的只是規則被破壞。

    天相vs文昌

    1. 基本上一個是主星(天相) 。
    2. 一個是輔星(文昌)，輔星會影響旁邊的主星。例如:貪狼+鸞喜+文昌，文昌會去影響貪狼的慾望。
    3. 文昌基本上為筆墨文書，背後指的是思慮，化忌時容易有情緒上的問題；天相不會有這種情況。
    4. 天相指的是我的生活周遭要被我所控制，因此天相的規範主要在人際關係上。
    5. 天相在雙星組合都是輔佐前面的星曜，讓前面那顆星曜較有規範、秩序。

    天相長相、特質

    瘦瘦高高、衣架子，比較不容易發胖(但受運限影響還是可能會發胖)、四肢細長(不論胖瘦)ex:林志玲、女生是美女、重視生活品味，會穿搭衣服。 天相星坐命遷線或子田線，很注重居家環境，會在意餐廳廁所是否乾淨。

    天相雙星的組合：

    1. 天相對宮紫破：我要的規則，在外面我希望能得到他人的尊重，讓人覺得我很有夢想、人緣好、人際關係佳。
    2. 紫破對宮天相：我自己希望得到尊重，並有各種想法，所以在人際關係方面希望大家都能聽我的。
    3. 紫殺對宮天府：這組需要自己奮鬥，而且人際關係可能不好，但他們不太在乎。
    4. 紫貪對宮空宮：這組通常生活過得不錯。紫貪被古書說的「一事無成」，是因為他們有能力「一事無成」。家境和外表通常不錯，但有時空宮會有煞星影響。
    5. 紫貪對拱：這組桃花很多，子位的貪狼隨時可能有煞星。這組是紫微系裡最需要三方四正有吉星的，因為這個人內心有很多慾望，若無足夠的條件（若無吉星，會覺得不管做何事，都心有餘而力不足），光有慾望只會很痛苦。
    6. 紫相對宮破軍：容易賺大錢。中州派稱為「造反之局」。紫微星很在乎尊嚴，希望大家按照他的規矩來，但在外面又表現得好像沒事。這種人在上班時若發現老闆不如自己，就容易自己去創業。對老闆來說是「造反」，但實際上是內心有夢想，想完成人生價值，所以會去做事，且愛面子，懂得規劃。只要運勢走得好，就很容易開創事業。
    7. 紫微系裡，紫殺、紫相、紫破若遇化權，落在命宮或財帛宮，尤其是命宮，都容易開創事業。
    8. 武相對宮破軍：這組容易與人產生金錢糾紛。因為無論在哪個宮位，只要運限命宮、財帛宮或六親宮位踩進去，都可能涉及金錢問題。

    武相、紫相、廉相，天相的差異：

    1. 紫微天相：在乎自己的自尊是否得到滿足，會在乎穿著打扮是否得體。
    2. 武曲天相：在乎他所訂定的規則和價值觀是否能帶來實質的利益。相對務實，不會那麼在意穿著打扮。如果武曲天相在命宮或財帛宮，而且有陀羅星，代表財務狀況需要特別注意。
    3. 廉貞天相：也會在乎穿著打扮。

    天相絕大多數的組合都非常好，只要沒有碰到煞星。 尤其是雙星組合，只要前面一顆星（無論是武曲、紫微或廉貞）碰到化祿或化權，都會相當不錯，因為他們會很清楚自己在做什麼，規則也掌握得很好。

    ★刑忌夾印格

    「刑忌夾印」指的是廉貞天相，刑指擎羊星、忌指天相化忌，這代表容易有官非。 但需要很多條件，並非單純有這些星曜就一定會犯官非，還要看【運限】。

    這樣的人可能會做一些別人不敢做或打破規則的事情，但這不一定會導致嚴重的官非，可能只是他們說的話比較容易有變化。

    ★蔭祿夾印格

    蔭指天梁星，祿指天梁化祿或巨門化祿，印則指天相星。 這個格局指的是天梁星化祿時，運限走到屬於「蔭祿夾印格」，會賺大錢。

    但這必須符合兩個條件： 第一，要看運限； 第二，不能有其他煞星，所以也不容易遇到。

    武曲破軍、對宮天相

    指的是花錢很浪漫。 這個組合如果有陀羅星走到巳或亥位，無論在哪個宮位，都要注意。 例如，陀羅星走到僕役宮，那一年就要注意與朋友金錢往來的問題。

    武曲破軍在子女宮，比較容易對小孩子出手大方。

    如果反過來子女宮是天相加陀羅，對孩子可能會大方，但這個大方的背後，會希望小孩要聽話。(2:01:23)

    補充 :

    • 武曲很務實的價值，就是金錢。
    • 太陽就是要聽他的話。
    • 巨門要看太陽的旺落：太陽旺位的巨門比較侃侃而談，落陷的巨門反而不會。
    • 天相星在正常情況下，會覺得人際關係好、幽默、好相處；但如果碰到運限的化忌、擎羊或陀羅，就會變得不好相處。

    廉貞星

    1. 五行屬陰火，化氣為「囚」。這個「囚」指的是「約束」，對自己的約束。廉貞星為官祿主。
    2. 被視為外交官的星曜，會重視是否有更多的發展機會。
    3. 聰明、反應快、邏輯好、不擇手段(背後真正的原因是不太在乎社會規則，當然就會容易踩到一些線)、行走的發電機、幽默、人緣好以及長得帥。(2:54:49)
    4. 廉貞清白格：指廉貞化祿，因約束之後而得到、增加了好處，此為正式的廉貞清白格。(加祿存是「乘旺主星」，就是把廉貞星的特色表現得很好)
    5. 廉貞星只要跟桃花星放在一起，就會變成桃花星。
    6. 五鬼星：廉貞是我們的靈魂，因此，廉貞星在命宮或福德宮比較容易跟另一個世界有接觸(比較容易，不等於一定會)。其中，比較會有感覺的是廉貞天相、廉貞破軍、廉貞貪狼組合的。
    7. 廉貞與七殺、破軍、貪狼同宮且帶煞忌時(解開約束)，殺破狼的特質(人類的慾望)就會大幅度的發揮出來。
    8. 廉貞七殺、廉貞破軍這種奔放的組合，只要加上煞星，通常磁場會非常強大。所謂磁場強大，是指會去影響到別人的磁場。
    9. 在人群裡面容易被看見，譬如廉貞系、太陽系、紫微系的人。

    廉貞星組合

    1. 廉貞七殺、對宮天府

      希望自己會很不一樣、有發展，然後會約束自己，並且很堅持。遷移宮是天府星，希望大家要聽他的，適合當將軍、或是軍師。 如果遇到煞星，當運限走得不對，就容易變成是黑道老大或地下錢莊的老闆(大家要聽他的，然後要有各種人際關係，內心有各種精密的盤算。)

    2. 廉貞破軍、對宮天相

      在甲年時，破軍會化權，廉貞會化祿，並且剛好有一隻擎羊放在天相星旁邊(卯位)。有夢想，而且用夢想取得權力，並且控制好自己，但在外關係較複雜(遷移宮是天相)。聰明有能力、長得帥，然後夢想會落實，並且敢有效率的做壞事。

    3. 廉貞天府，對宮七殺

      唯一能控制廉貞的是天府星，不論廉貞是否化祿，因為天府有很好的掌控能力(天府化煞為用，但不能超過2顆)。

    4. 廉貞天相，對宮破軍

      加煞忌，會去做一些以前不敢做的事。例如：郭台銘在那個大限到處提告、或像愛莉莎莎。

    逢天府時要看天相的狀況；看天相時也要看天府星的狀況。

    看天府時要看天相，看天相時也要看天府，因為剛好會在三合內。簡單來講，如果天相星要做什麼犯規的事情，天府星會去控制它；但如果天府星也帶了煞，那就會不好控制。
